Add-On Services — Roadmap

📊 Embed Analytics

Track views per menu slot — how many guests scanned or loaded each menu. Shows restaurants their ROI directly in dashboard.

Pro+ · High priority

📱 Social Auto-Post

When a menu publishes or a holiday promo goes live, auto-post to Instagram and Facebook via Make.com webhook.

Pro+ · Medium priority

💬 SMS Menu Link

Send the menu URL via text message to customers. Integrates with Twilio. Great for table QR fallback.

Pro+ · Medium priority

🌐 White-Label Embed

Custom domain for embed URL (menu.yourrestaurant.com instead of servvee.online/embed/...). Requires DNS config.

Agency · Low priority

👥 Multi-User / Team

Multiple staff logins per restaurant account with role-based access (owner vs. manager vs. designer).

Agency · Medium priority

🌍 Menu Translations

AI auto-translate menu content into Spanish, French, Portuguese etc. Display language based on visitor browser locale.

Add-on · Future

🔔 Outbound Webhooks

Push events (menu published, promo went live, promo ended) to Make.com / Zapier / custom endpoints.

Pro+ · Low priority

🖨 Print-Ready Export

Export the current menu as a print-ready PDF at 300 DPI. Useful for physical menu boards and table cards.
